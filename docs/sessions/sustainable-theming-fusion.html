Sustainable Theming with Fusion

Session evaluation:
Evaluate this session
Speaker(s):
Stephanie Pakrul, Sheena Donnelly

Don't theme yourself into a corner. We built Fusion after years of feedback from designers, site builders, clients, and developers. It's a powerful and supported base theme, with layout and style configuration options built in that you (or your clients) can control through Drupal's UI using the Skinr module. Based on a simplified 960px or fluid 12 or 16-column grid, you can easily define your own custom grid if needed.

In this session we'll cover the basics of Fusion from a themer's point of view, and how it makes themes more sustainable and makes your work flow more efficient. We'll show you how to get started using Fusion as a base theme, how to use the collapsible regions and theme settings to set up the layout you need, and how to begin creating more sustainable themes using Skinr.
