Modern Web Typography

Session evaluation:
Evaluate this session
Speaker(s):
Scott Reynen, Ken Woodworth

Video by Variant Studios

Today's browsers are no longer limited to Arial and Times New Roman for fonts. You can now safely use nearly any font you want on your websites. We'll give you everything you need to get started with web typography in Drupal, including tools, techniques, legal issues, and design considerations.
